# 340 Livingston — SBJ Group

A minimal full-screen slideshow of the 340 Livingston (Brooklyn, NY) render studies
for SBJ Group Architects & Interior Designers.

- **Live:** https://amlervishal.github.io/340-livingston/
- One photo per screen. Navigate: arrow keys / spacebar, click left–right, or swipe (mobile).
- Cover + 30 renders. Black / Syne, styled after the SBJ deck cover.

## Reorder / edit slides
Slide order is the `IMAGES` array near the top of the `<script>` in `index.html`.
Reorder or remove entries; add a new file to `images/` and its name to the array.

Images are web-optimized JPEGs (~1600px). Source renders live in the rendering-workflow project.
